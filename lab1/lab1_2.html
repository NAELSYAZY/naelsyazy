Гиперссылка — часть гипертекстового документа, которая указывает на другой элемент/файл/страницу. Обычно ссылки выделяются цветом и меняют стиль при наведении. :contentReference[oaicite:1]{index=1}

Как сделать ссылку

Ссылка создаётся тегом <a>. Главный атрибут — href, в нём указывается адрес (URL). :contentReference[oaicite:2]{index=2}

Абсолютная ссылка

Абсолютные ссылки начинаются с протокола (часто https://).

<p>
  <a href="https://ya.ru">Поисковая система Яндекс</a>
</p>

Относительная ссылка

Относительные ссылки строятся относительно текущей страницы/папки: images/pic.jpg, ../help/me.html и т.д. :contentReference[oaicite:3]{index=3} 

<p><a href="images/xxx.jpg">Посмотрите на мою фотографию!</a></p>
<p><a href="tip.html">Как сделать такое же фото?</a></p>

Атрибут target

По умолчанию ссылка открывается в текущей вкладке. Атрибут target позволяет изменить поведение: _blank, _self, _parent, _top. :contentReference[oaicite:4]{index=4}

Открытие в новой вкладке

Если используете target="_blank", добавляйте rel="noopener noreferrer". :contentReference[oaicite:5]{index=5} 

<a href="https://webref.ru"
   target="_blank"
   rel="noopener noreferrer">
  Ссылка откроется в новой вкладке
</a>

Тег base (чтобы все ссылки по умолчанию открывались в новой вкладке)

<head>
  ...
  <base target="_blank">
</head>

Пример ссылки, которая в таком случае всё равно откроется в текущем окне:

<a href="page.html" target="_self">Открыть в текущей вкладке</a>

Изображение в качестве ссылки

В качестве ссылки можно использовать изображение: картинка помещается внутрь <a>...</a>. У изображения обязательно должен быть alt. :contentReference[oaicite:6]{index=6}

Пример (кликабельная картинка)

Ниже — рабочий пример (внешняя картинка). Вы можете заменить src на свою из папки images/.

Код примера

<a href="sample.html">
  <img src="images/sample.gif" width="50" height="50" alt="Пример">
</a>

Если вокруг изображения появляется рамка, её можно убрать через CSS (например, a img { border: none; }). :contentReference[oaicite:7]{index=7}

Ссылки внутри страницы

Для внутренней навигации используют якоря (закладки) — это элементы с уникальным id. Переход делается ссылкой вида href="#имя". :contentReference[oaicite:8]{index=8}

Пример якоря и ссылки

<p id="top"></p>
...
<p><a href="#top">Наверх</a></p>

Переход к якорю на другом сайте

<a href="https://ru.wikipedia.org/wiki/HTML5#HTML_5.1">HTML 5.1</a>

Если ссылка ведёт на якорь, которого нет, ошибка обычно не показывается — переход просто не произойдёт. :contentReference[oaicite:9]{index=9}
